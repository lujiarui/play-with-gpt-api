# play-with-gpt-api

## Dependency

```bash
pip install openai

# Acquire your API keys from https://platform.openai.com/account/api-keys
export OPENAI_API_KEY="<acquired keys>"

echo "export OPENAI_API_KEY=${OPENAI_API_KEY}" >> ~/.bashrc # if you use bash in your shell
```

Now you're all set. You can begin using GPT-4 on your macOS.
